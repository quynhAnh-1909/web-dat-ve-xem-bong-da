# web-dat-ve-xem-bong-da
Website đặt vé xem bóng đá CLB Hoàng Anh Gia Lai được xây dựng bằng Java (Servlet/JSP) theo mô hình MVC, nhằm hỗ trợ người hâm mộ xem lịch thi đấu và đặt vé trực tuyến một cách nhanh chóng và tiện lợi.
Xây dựng một website đặt vé bóng đá dành riêng cho CLB Hoàng Anh Gia Lai.
Cung cấp các chức năng cơ bản cho người dùng như xem lịch thi đấu, chọn chỗ ngồi, đặt vé và xem lại vé đã mua, thanh toán trực tuyến.
